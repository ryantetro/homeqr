# SQL Scripts

This folder contains SQL scripts for database operations, migrations, and utilities.

## Files

- **`apply-ai-columns-now.sql`** - Apply AI enhancement columns to database
- **`apply-ai-columns-simple.sql`** - Simplified version of AI columns migration
- **`check-listings-table.sql`** - Query to check listings table structure
- **`debug_url_queries.sql`** - Debug queries for URL-related issues
- **`verify-ai-columns.sql`** - Verify AI columns exist and are correct
- **`verify-migration.sql`** - Verify database migrations have been applied

## Usage

These scripts are typically run in the Supabase SQL Editor or via the Supabase CLI.

**Note:** The main database schema and migrations are located in the [`supabase/`](../supabase/) folder.
